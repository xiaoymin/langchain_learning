## 安装仓库


1、虚拟化Python环境

```shell
pip install virtualenv
```


2、创建环境

```shell
virtualenv env
```

3、安装依赖包

```python
pip install -r requirements.txt
```

4、更新某个具体版本的包，例如`LangChain`

```shell
pip install -upgrade langchain=0.0.213
```
